# -Reducao-de-Dimensionalidade-em-Imagens-para-Redes-Neurais

**🎯 Descrição do Projeto:**

Este projeto tem como objetivo demonstrar a redução de dimensionalidade em imagens utilizando Python. A aplicação realiza a conversão de uma imagem colorida para: 
- Escala de cinza (0 a 255)
- Imagem binarizada (0 e 255)

---

**💻 Recursos Usados:**

- Python
- OpenCV
- Unsplash

---

**🧠 Conhecimentos Aplicados:**

- Redução de Dimensionalidade
- Python aplicado à Visão Computacional
- OpenCV na prática
